Para la ejecución de las pruebas es necesario haber clonado el repositorio previamente con el comando en el cmd:

git clone https://github.com/Riverapjj/QA-BP-Devsu.git
cd demoblaze-e2e



2. Luego de abrir el proyecto, abrimos un command prompt en el IDE utilizado para instalar las dependencias del framework con el siguiente comando:

mvn clean install



3. Una vez instaladas las dependencias, ejecutamos el test con el comando:

mvn clean test
